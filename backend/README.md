# Shopify Custom Product App - Backend (Phase 1)

This directory will contain the backend services required for the Shopify Custom Product App.

## Technology Stack
- **Language**: Python (FastAPI) or Node.js (Express/Remix) - *Python implied by project context*
- **Database**: PostgreSQL / MySQL

## Responsibilities
1.  **Configuration API**: Save/Load merchant product settings (Print areas, base images).
2.  **Design API**: Receive and store design JSON from the Frontend Editor.
3.  **App Bridge Auth**: Handle Shopify Session Tokens.
4.  **Webhooks**: Process `orders/create` to link designs to orders.

## Status
- Currently initialized as a placeholder.
- Core logic to be implemented in subsequent steps.
